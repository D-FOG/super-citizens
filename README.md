# Supersite Citizens

Modern multi-page Next.js website for a structured Kingdom growth system.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn-style primitives
- Next.js route handlers for joins, cluster applications, contact, and Flutterwave-ready payments

## Run

```bash
npm install
npm run dev
```

Dark mode is the default. Theme choice is persisted with `localStorage`.
